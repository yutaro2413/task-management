# GitHubへのプッシュ手順

## 🚀 プッシュ方法

### 方法1: ターミナルで直接実行（推奨）

ターミナル（PowerShellまたはコマンドプロンプト）で以下を実行：

```bash
git push -u origin main
```

認証を求められたら：
- **Username**: `yutaro2413`
- **Password**: 作成したPATを貼り付け

### 方法2: PATを環境変数に設定（オプション）

PowerShellで：

```powershell
$env:GIT_ASKPASS = "echo"
git -c credential.helper='!f() { echo "username=yutaro2413"; echo "password=YOUR_PAT_HERE"; }; f' push -u origin main
```

（`YOUR_PAT_HERE`を実際のPATに置き換える）

### 方法3: GitHub CLIを使用（推奨）

GitHub CLIがインストールされている場合：

```bash
gh auth login
git push -u origin main
```

## ✅ 成功したら

GitHubのリポジトリページ（`https://github.com/yutaro2413/task-management`）でファイルが表示されているか確認してください。

## 📝 今後の使い方

一度プッシュが成功すれば、今後は以下のコマンドで簡単にプッシュできます：

```bash
git add .
git commit -m "コミットメッセージ"
git push
```

認証情報はWindows Credential Managerに保存されているので、再度入力する必要はありません。
