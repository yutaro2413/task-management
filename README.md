# 📋 タスク整理・思考の整理

週次タスク管理と思考の整理を行うためのリポジトリです。

## 📁 ファイル構成

- `タスク整理・思考の整理.md` - メインのタスク管理ファイル（週次更新）
- `週次タスク_テンプレート.md` - 週単位でファイルを作成する際のテンプレート

## 🎯 使い方

### 週の初め
1. 今週の計画セクションに目標とタスクを記入
2. Mermaidガントチャートにタスクを追加
3. 詳細タスク管理テーブルを更新

### 毎日
1. タスクの進捗を更新
2. 新しいタスクを追加
3. 完了したタスクを「完了したタスク」セクションに移動

### 週の終わり/翌週初め
1. 週次レビューセクションで振り返り
2. 来週の計画を立てる
3. 必要に応じて新しい週次ファイルを作成

## 📊 機能

- **ガントチャート**: Mermaid形式で週間タスクを視覚化
- **WBS**: 作業分解構造でタスクを階層化
- **優先順位管理**: 緊急度・重要度に基づくタスク分類
- **思考の整理**: 気づき、課題、アイデアを記録
- **週次レビュー**: 振り返りと次週計画

## 🔄 バージョン管理

このリポジトリはGitHubで管理しており、以下のメリットがあります：

- ✅ 変更履歴の追跡
- ✅ 週次レビューやタスク進捗の記録
- ✅ バックアップとしての機能
- ✅ 過去の計画や振り返りの参照

## 📝 コミットメッセージの例

- `feat: 週次タスク追加 (2024/01/24-01/30)`
- `update: タスク進捗更新 - 注文内容API連携完了`
- `docs: 週次レビュー追加 - 2024/01/24週`

## 🛠️ セットアップ

```bash
# Gitリポジトリの初期化（まだの場合）
git init

# リモートリポジトリの追加（GitHubで作成後）
git remote add origin https://github.com/your-username/your-repo-name.git

# 初回コミット
git add .
git commit -m "feat: 初期セットアップ - タスク整理システム"

# GitHubにプッシュ
git push -u origin main
```
